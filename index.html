2. Dinâmica de Mapeamento Inicial

Levantamento das percepções que os alunos têm de certos conceitos conectados aos temas desenvolvidos no curso com o objetivo de estabelecer marco zero e mapeamento de expectativas.

Materiais:

  • Post-it (5 cores - um para cada pergunta)
  • Mural (5 folhas de flip chart com as palavras conceitos escritas
  • Canetas

Instruções de facilitação:

  • Apresentar aos alunos que você fará 3 rodadas de perguntas sobre o que eles entendem hoje sobre as palavras conceitos.
  • Após cada rodada cada aluno lerá seu post-it para o grupo e colará no mural que terá a palavra escrita
  • Oriente os alunos a serem ágeis, escrever o que vem mesmo na hora, na cabeça ao escutarem a palavra.
  • Ao fim de cada rodada a facilitadora faz uma fala sobre o que os alunos trouxerem de cada conceito.

3. Vivência LED+Bateria

Primeiro contato com dois elementos simples na área de eletrônica: um LED e uma bateria de modo a propor uma investigação da interface de hardware/produto.

Materiais:

  • Bateria CR2032 (uma para cada aluno)
  • LED 3 mm difuso (um para cada aluno)

Instruções de facilitação:

  • Evitar de falar o nome destes dois elementos (LED e bateria). A proposta é investigar o que eles acham que são estes elementos e que eles digam os nomes.
  • Perguntar aos alunos: Que elementos são estes? Como eles se apresentam ou se comportam?

3. Vivência LED+Bateria

Exploração técnica

Questões levantadas de forma rápida, permitindo algumas respostas mas sem alongar muito a discussão. O objetivo é despertar a reflexão. Algumas questões podem ser encaminhadas para pesquisa fora do tempo de aula.

Exploração conceitual

Gatilhos para facilitadores:

Exploração conceitual pode ser também em tecnologia, inovação e wearables (a exploração conceitual foi feita a partir de insights das respostas dadas na dinâmica de mapeamento)

Exploração Técnica

Bateria

  • Que componente é este? O que fez identificar que é uma bateria?
  • O que significam as coisas escritas na embalagem?
  • Qual a diferença entre bateria e pilha?
  • Ouvir primeiras impressões (onde é utilizada? parece descartável?)

LED

  • Que componente é este?
  • O que significa o nome LED?
  • Pode ser ligado de qualquer forma (lado) na bateria?
  • Polaridade: como identificar os pólos > identificação imediata
  • Se cortar a terminação maior e deixá-las do mesmo tamanho, como identificar? (investigação de interface de hardware)

Exploração Conceitual

Vestuário

  • Por que nos vestimos?
  • O que é vestuário? Uma toalha de mesa é vestuário?
    Um óculos é vestuário?
  • Vestuário como proteção, vestuário como identidade,
    questões ambientais/culturais

Interfaces

  • O que é interface?
  • Como os produtos bateria e LED utilizaram da sua interface para saber o que eram?
  • Exemplo do celular: como sabemos usar os aplicativos?
  • Metáforas cognitivas > convenções culturais de usabilidade
  • Por que a interface é importante? > Comunicação com o usuário em produtos de inovação

5. Vivência LED no corpo

Prototipagem "Onde você colocaria o LED no seu corpo?" com diversos materiais disponíveis (chapéu, mão manequim, linhas, papel alumínio, arame, papéis, etc)

Materiais:

  • Bateria CR2032 (uma para cada aluno)
  • LED 3 mm difuso (um para cada aluno)
  • Materiais diversos: linha de costura, linha condutiva, bom-bril, cera de abelha, papel alumínio, bijouterias, etc.

Instruções de facilitação:

  • Perguntar aos alunos que produtos são estes? Como eles se apresentam ou se comportam?
  • Após cada rodada cada aluno lerá seu post-it para o grupo e colará no mural que terá a palavra escrita
  • Oriente os alunos a serem ágeis, escrever o que vem mesmo na hora, na cabeça ao escutarem a palavra.
